###### ΕΓΚΑΤΑΣΤΑΣΗ ######
Για να τρέξετε το πρόγραμμα σωστά, ακολουθήστε τα βήματα:

[ Πιθανότατα θα χρειαστεί να απενεργοποιήσετε το αντιικό σας πρόγραμμα, το οποίο θα σας προειδοποιήσει
εσφαλμένα ότι το πρόγραμμά μας είναι κακόβουλο - δεν είναι όμως!]

α) Αποσυμπιέστε τα αρχεία σε κάποιο φάκελο του υπολογιστή σας.
β) Βεβαιωθείτε πως στον αποσυμπιεσμένο φάκελο υπάρχουν όλα τα αρχεία (δείτε ειδικό παράρτημα).
γ) Βεβαιωθείτε πως έχετε κατεβασμένη την Python, έκδοση 3.7.x και πάνω.
δ) Εγκαταστήστε τις απαιτούμενες βιβλιοθήκες της Python (δείτε ειδικό παράρτημα).
ε) Τρέξτε το αρχείο source.py.

##### MODULES ΠΟΥ ΧΡΕΙΑΖΟΝΤΑΙ ΧΕΙΡΟΚΙΝΗΤΗ ΕΓΚΑΤΑΣΤΑΣΗ ######

α) xlwings (*)
β) Pillow
γ) tkcalendar
δ) bbmod (βρίσκεται στα αρχεία του προγράμματος, μην την κατεβάσετε μέσω του PyPI).

(*) To module pythoncom περιλαμβάνεται στο package της xlwings. Επίσης, υπάρχει η πιθανότητα τρέχοντας το πρόγραμμα να προκύψει το εξής
ModuleNotFound Error: pywin32 not found. Απλώς επανεκκινήστε το IDLE αφού εγκαταστήσετε την xlwings.

##### ΕΓΚΑΤΑΣΤΑΣΗ MODULES #####

[Για να ελέγξετε αν έχετε ήδη εγκαταστήσει κάποιο module - βιβλιοθήκη, ακολουθήστε τα παρακάτω βήματα μέχρι
και το β) και μετά πληκτρολογήστε «pip freeze.»

Αφού εγκαταστήσετε το PyPI (pip):

α) Πατήστε [WIN] + [R]
β) Πληκτρολογήστε «cmd» ή «powershell» και πατήστε [ENTER]
γ) Πληκτρολογήστε «pip install "το_όνομα_του_module"»
